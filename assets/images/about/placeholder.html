Imágenes Necesarias para la Página About

La página de About requiere las siguientes imágenes que faltan actualmente en el proyecto:

1. agency-hero.jpg

Esta imagen se usa como fondo en la sección hero de la página About.

Ruta: /assets/images/about/agency-hero.jpg

Sugerencia: Usa una imagen de una oficina creativa o moderna. Puedes descargar la siguiente imagen gratuita de Unsplash:

Descargar Imagen Sugerida

2. fallback-hero.jpg

Esta imagen se usa como respaldo (fallback) en caso de que la imagen principal no se cargue correctamente.

Ruta: /assets/images/about/fallback-hero.jpg

Sugerencia: Usa una imagen similar a la principal pero más ligera. Puedes descargar la siguiente imagen gratuita de Unsplash:

Descargar Imagen Sugerida

Instrucciones para implementar estas imágenes:

  1. Descarga las imágenes haciendo clic en los botones de arriba
  2. Renombra las imágenes descargadas como agency-hero.jpg y fallback-hero.jpg
  3. Coloca ambas imágenes en la carpeta /public/assets/images/about/ de tu proyecto

Una vez implementadas estas imágenes, la página About mostrará correctamente todos los elementos visuales sin errores 404.
